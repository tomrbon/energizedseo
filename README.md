# EnergizedSEO

Professional websites with built-in SEO for local businesses.

## Development

```bash
npm install
npm run serve
```

## Build

```bash
npm run build
```

Output goes to `_site/` directory.

## Deployment

This site is configured for Cloudflare Pages deployment. Connect your GitHub repo to Cloudflare Pages and it will auto-deploy on push to main branch.

### Build Settings for Cloudflare Pages

- **Build command:** `npm run build`
- **Build output directory:** `_site`
- **Node version:** 18 (or higher)

## Tech Stack

- [11ty (Eleventy)](https://www.11ty.dev/) - Static site generator
- Vanilla CSS - No framework, optimized performance
- Cloudflare Pages - Hosting & CDN
